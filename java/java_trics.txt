Для улучшения перфоманса приложения:
- использовать несинхронизированные коллекции, где это возможно (если возможно массивы)
- указывать изначальный размер коллекции или стрингбилдера
- минимизировать использования instanceOf
- не использовать вложенные циклы
- избегать sysout, использовать логгеры
- минимизировать использования Date и Calendar
- минимизировать вызов JNI методов
